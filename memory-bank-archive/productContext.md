## Project Goal

* Create a sophisticated wedding website with teal & silver accents, inspired by fancy restaurant menus with rich, textured, and elevated design elements.
* Gather input from visitors through a wedding location poll and photo sharing functionality.
* Provide comprehensive information about the wedding, including the couple's story, event details, travel info, registry, and RSVP system.

## Key Features

* **MVP (2-Day Timeline):**
  * Poll functionality for wedding location (England, LA, Minnesota, Other)
  * Photo sharing with Google Albums integration
  * Admin approval system for comments and photos
  * Email notifications for submissions
  * Basic "Our Story" section
  * Elegant design with teal & silver accents

* **Full Features (60-Day Timeline):**
  * Enhanced design with rich textures and brocaded elements
  * Complete "Our Story" section
  * Event details and travel information
  * Registry links
  * RSVP system with Google Forms integration
  * Advanced photo gallery with lightbox and slideshow
  * Performance optimization and final polishing

## Overall Architecture

* **Frontend:** HTML5, CSS3 (with Sass/SCSS), JavaScript (ES6+)
* **Hosting:** GitHub Pages
* **Backend Services:**
  * Form processing with serverless functions
  * Email notification system
  * Content approval system
  * Google Albums integration
